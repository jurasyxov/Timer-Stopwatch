# ⏱ Timer / Stopwatch

A simple stopwatch implementation written in multiple programming languages.

---

## 📌 Overview

This repository demonstrates how to create a basic stopwatch that counts elapsed time in:

- JavaScript
- Python
- C#

---

## 🚀 Features

- Start counting from zero
- Updates every second
- Lightweight implementation
- Easy to understand

---

## 📂 Project Structure

```text
javascript/
python/
csharp/
```

---

## Example Output

```text
Elapsed: 1s
Elapsed: 2s
Elapsed: 3s
Elapsed: 4s
```

---

## Purpose

The purpose of this repository is educational:

- Compare implementations across languages
- Practice basic timing functions
- Learn language syntax

---

## Requirements

### JavaScript

Node.js

### Python

Python 3.x

### C#

.NET SDK

---

## Run

### JavaScript

```bash
node stopwatch.js
```

### Python

```bash
python stopwatch.py
```

### C#

```bash
dotnet run
```
